# SIM A — Basic Underwriting Behaviour

## Purpose
SIM A introduces the simplest underwriting environment used in the
marine insurance simulation program.

The objective is to examine how underwriting decisions evolve across
time when agents observe stochastic loss outcomes.

## Institutional Layer Introduced
Individual underwriting behaviour.

## Model Structure

- Single underwriting agent
- Voyage risks generated each period
- Stochastic loss distribution
- Sequential underwriting periods

## Key Mechanisms

- underwriting decision per voyage
- stochastic loss generation
- cumulative performance tracking

## Simulation Environment

- one underwriting agent
- repeated voyage opportunities
- stochastic loss events

## Outcome Metrics

- cumulative profit
- loss experience
- underwriting performance across time

## Relationship to Previous Simulation

This is the initial simulation in the series.

## Notes

The model intentionally excludes capital constraints and institutional
mechanisms in order to isolate basic underwriting behaviour.
